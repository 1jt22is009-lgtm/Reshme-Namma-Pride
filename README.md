# Reshme-Namma Pride 🧶

**Reshme-Namma Pride** is a production-ready Android application designed for Karnataka sericulture. It provides farmers with live supervision, climate tracking, and a real-time silk price index.

## Features ✨

- **Dashboard**: Live rearing command center with climate status (Optimal, Warning, Danger).
- **Climate Monitoring**: Track temperature and humidity with actionable advice for silk rearing.
- **Silk Market Index**: Real-time price tracking for various silk varieties (Bivoltine White, Cross Breed, Mulberry Raw) with historical line charts.
- **Batch Management**: Start and track rearing batches from start to harvest.
- **Modern UI**: Built with Jetpack Compose, featuring a premium glassmorphism theme.

## Tech Stack 🛠️

- **Language**: Kotlin
- **UI**: Jetpack Compose
- **Database**: Room (Offline-first architecture)
- **Architecture**: MVVM with Repository pattern
- **Charts**: Custom Canvas-based line charts
- **Background Tasks**: WorkManager
- **Navigation**: Compose Navigation

## Setup & Installation 🚀

1. Clone this repository.
2. Open the project in Android Studio.
3. Sync Gradle and build the app.
4. Run on an emulator or physical device.
